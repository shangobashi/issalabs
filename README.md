# ISSALABS

Static marketing site for `issalabs.xyz`.

## Local Development

```powershell
npm.cmd run dev
```

## Deployment

- Vercel project: `issalabs-preview`
- Custom domains:
  - `https://issalabs.xyz`
  - `https://www.issalabs.xyz`

## Notes

- This site is intentionally lightweight and static.
- Security headers are managed through `vercel.json`.
